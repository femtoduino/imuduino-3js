IMUduino examples

Here are a bunch of examples of using the IMUduino with javascript to give you a taste of what you can do with this device

Each section below is associated with a specific sketch, either from the official IMUduino github, or from my sketches folder. A link will be provided for each section.

To use a sample, load the appropriate sketch, run npm config set imuduino-3js:type [name of section], restart the server, and then click on the link for the sample.

IMUduino_Bluetooth_UART_YawPitchRoll

These use the IMUduino_Bluetooth_UART_YawPitchRoll sketch from the official IMUduino repo

Pitch-Yaw-Roll cube
This is a very simple setup that allows you to control the rotation of a cube by rotating the IMUduino

IMUduino_Bluetooth_UART_AccelXYZ

Load the IMUduino_Bluetooth_UART_AccelXYZ sketch firmware onto your board.

Accel XYZ data stream
This example streams the Accelerometer XYZ values as accel_x, accel_y, accel_z

IMUduino_Bluetooth_UART_GyroXYZ

Load the IMUduino_Bluetooth_UART_GyroXYZ sketch firmware onto your board.

Gyro XYZ data stream
This example streams the Gyroscope XYZ values as gyro_x, gyro_y, gyro_z

IMUduino_Bluetooth_UART_MagXYZ

Load the IMUduino_Bluetooth_UART_MagXYZ sketch firmware onto your board.

Mag XYZ data stream
This example streams the Magnetometer XYZ values as mag_x, mag_y, mag_z

IMUduino_Bluetooth_UART_TempAlt

Load the IMUduino_Bluetooth_UART_TempAlt sketch firmware onto your board.

Temp/Alt data stream
This example streams the Barometer's Temperature and Altitude values as baro_temp, baro_alt
